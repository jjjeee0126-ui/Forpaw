## D-001 | 앱 컨셉 확정
**날짜:** 2026-03-21  
**결정:** 반려동물 사진 → 털 색상 기반 하트 모양 캐릭터 생성  
**근거:** 영진님 레퍼런스 이미지 5장 (Sunshine, Ari, Gureum, Sangsil, Bam)  
**색상 규칙:**
- 하트 털 색 = 동물 털 주색상
- 이름 글자 색 = 포인트 색 (눈 색 or 털 포인트)
- 배경 = 포인트 색 계열 그라데이션

## D-002 | 수익 모델 — AI 생성 비용 제거
**날짜:** 2026-03-21  
**결정:** AI 이미지 생성(DALL-E 등) 사용 안 함. Canvas API로 직접 렌더링.  
**근거:** 이미지 생성 1장 ≥ 14원, 리워드 광고 1회 ≤ 5원 → 광고만으론 구조적 적자  
**대안:** Canvas pixel sampling으로 털 색 추출 → Canvas로 하트+털 직접 그리기 → 비용 0원  
**광고 수익화:** 배경 테마 언락 (리워드 광고 시청)  
**폐기된 안:**
- ~~첫 생성만 AI (DALL-E 3)~~ → 비용 발생
- ~~가챠 구조~~ → 미리 생성 비용 발생
