## 핵심 컨셉 (확정)

동물 사진 + 이름 입력 → Vision AI가 털 색상/재질/포인트 색상 분석 → **하트 모양 털뭉치 캐릭터** 생성.

### 색상 규칙
- **하트 털 색** = 동물 털 주색상
- **이름 글자 색** = 포인트 색 (눈 색 or 털 포인트)
- **배경** = 포인트 색 계열 그라데이션

### 레퍼런스 예시
| 이름 | 털 | 이름 색 | 배경 |
|------|-----|---------|------|
| Sunshine | 황금색 짧은털 | 핑크 | 파랑-보라 그라데이션 |
| Ari | 회청색 털 | 주황 | 노란색 |
| Gureum | 흰/민트 곱슬털 | 파랑 | 분홍 |
| Sangsil | 하얀 복슬털 | 핑크 | 민트-파랑 그라데이션 |
| Bam | 검정 털 | 민트 | 민트 그라데이션 |

## 비용 구조
- 이미지 생성 1장: 최소 14원
- 리워드 광고 1회: 최대 5원
- → AI 생성 횟수 최소화가 핵심

### 채택 방향: Canvas 기반 렌더링 (AI 생성 없음)
- Vision API로 털 색 분석 ($0.002) or Canvas pixel sampling
- **하트 + 털 질감은 Canvas로 직접 렌더링** → 생성 비용 0원
- 광고 시청으로 배경 테마 언락

## 프로토타입 현황

- **코드 위치:** `~/Projects/for-paw`
- **기술 스택:** React + Vite + Canvas API, 모바일(390px) 기준
- **상태:** Ollama(qwen3:32b) 코드 생성 중 (2026-03-21 18:10)
- **실행:** `cd ~/Projects/for-paw && npm run dev`

### 구현 범위
1. **홈** — mock 카드 목록 + "새 친구 등록" 버튼
2. **등록 플로우 4단계** — 사진 업로드 → 이름 입력 → 분석 로딩 → 캐릭터 공개
3. **꾸미기** — 배경 테마 8종 (CSS 프리셋), 잠긴 테마는 광고 모달(5초) 후 언락
4. **상세** — 저장/공유

### 기술 구현
- Canvas API로 하트 모양 + 털 질감 렌더링
- 업로드 이미지에서 canvas pixel sampling으로 dominant color 추출
- 추출 색상 → 배경 그라데이션 자동 생성 (보색 계산)
- React Router 페이지 전환
